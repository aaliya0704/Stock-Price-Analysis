# 📈 Stock Price Visualizer

## Project Overview

This project downloads historical stock market data using the yfinance library and performs basic time-series analysis.

The project visualizes stock prices over time and calculates moving averages to identify trends.

## Technologies Used

- Python
- Pandas
- Matplotlib
- yfinance
- Jupyter Notebook

## Features

- Download stock data
- Plot closing prices
- Calculate moving averages
- Visualize market trends
